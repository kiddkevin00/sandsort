Sand
sort.

A small pour-and-sort puzzle for iPhone. Tip jars of colored sand into each other until every jar is one color. No ads, no IAP, no clock.

v1.0.0·iPhone · iOS 16+

01

Overview

How to play

Tap a jar to pick it up — the jar lifts. Tap another jar to pour the top color of the picked-up jar into it. You can only pour onto an empty jar or onto sand of the same color. The whole top run pours at once, or as much as fits.

  • Each level finishes when every jar is one color (or empty).
  • Undo walks back the move history one step at a time.
  • Restart resets the same deal; New deal reshuffles the same level.
  • Best move count per level is recorded — fewer is better.
02

Origin

Is this a clone of something?

No. The pour-and-sort mechanic is a long-running puzzle pattern with many independent implementations (Water Sort, Ball Sort, Sand Sort, etc.). Sandsort is an original implementation with its own UI, code, color palette, and branding — no assets, naming, or layouts from any specific app are reused.

03

Levels

How does difficulty progress?

Levels add more colors and remove spare jars. Each new level unlocks once you clear the previous one. From the menu you can jump back to any unlocked level — your best move count for each one is shown there.

04

Solvability

What if I get stuck?

Sandsort tells you when there are no pours available. Undo walks back the bad move; Restart resets the deal; New deal reshuffles. Every shuffle is generated to avoid trivially-sorted openings.
