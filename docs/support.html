Support

Frequently asked, honestly answered.

If your question isn't here, open an issue on GitHub. Public, traceable, and the same channel I use for everything else.

Is this a clone of any specific app?

No. The pour-and-sort mechanic (Water Sort / Ball Sort / Sand Sort) is a generic puzzle pattern with many independent implementations. Sandsort is an original take with its own UI, palette, level curve, and code.

How are levels generated?

Each level mixes a fixed number of colors (3–8) into jars of capacity 4, then shuffles. The generator reshuffles up to eight times if a jar comes up trivially already-sorted, to avoid boring openings.

What does New layout do?

Reshuffles the current level with a fresh seed. Same level config (same number of colors and jars), different starting arrangement.

What if I get into an impossible state?

The board shows a 'No pours available' banner. Tap Undo to walk back, Restart to reset the deal, or New layout to reshuffle.

Is anything sent anywhere?

No. Current level, unlocked levels, best move counts, and the haptics setting are stored on this device only via AsyncStorage.
